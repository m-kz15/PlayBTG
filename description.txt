【このゲームについて】

    ・このWebゲームは制作者が修了課題として制作したものとなっており、
    　はじめてのWiiという作品の「タンク」というゲームを参考にして作られています。

    ・このゲームのシステムの約９割はEnchant.jsを用いたJavaScriptで作られています。

    ・このゲームで遊ぶ際はサーバが必要となります。
    　xamppなどのローカルサーバをご用意下さい。



【ゲームのルールについて】

    ・ゲームのクリア条件    => 敵を全滅すること。

    ・ゲームの敗北条件      => プレイヤーが倒されること
    
    ・プレイヤー及び敵の砲弾について
        => 各戦車ごとに弾速や弾数、反射回数などが用意されており、
        　 敵は[/stage/___.js]に、
        　 プレイヤーは「BTG.js」の戦車生成処理で設定されています。



【ゲームの操作について】

    ・移動      => WASDキー

    ・砲の照準  => マウス操作 or 十字キー

    ・砲撃      => 左クリック or スペースキー

    ・爆弾設置  => Qキー



【ゲームのシステム制御について】

    ・ゲームの起動から物理判定等のあらゆる制御は「BTG.js」が行っています。

    ・プレイヤーはPlayerクラス、敵(弱)はNewAiクラス、敵(強)はEliteクラス、
    　ボスはBossクラスで生成、制御を行っています。

    ・弾はBulletクラス、BulletColクラスがあり、
    　描画や戦車との衝突判定をBulletクラスが担い、
    　物理演算処理をBulletColクラスで構成されています。
    　（BulletAimやFire、Smokeクラスは補助的存在）

    ・ある程度のバグは改善してきましたが、壁へのめり込みや弾の判定すりぬけ、
    　ステージ左上で発生する敵戦車の不審な挙動などがありますが、
    　今後修正していくのでどうかご了承ください。



【ゲーム内の画像や音声について】

    ・ゲーム内で使用される画像は[image]フォルダに格納されています。
    　新しく追加したい場合は[image]の該当フォルダに入れてください。

        戦車や弾などの画像  => [/image/ObjectImage/]
        マップ画像          => [/image/MapImage/]

    ・Enchant.jsで必須となる画像については[enchant/image]に格納されています。

    ・ゲーム内で使用される音声は[sound]フォルダに格納されています。



【ステージについて】

    ・ステージの総数は２０。(変動あり)
    【2023/12/15追加】
    　ステージの総数が23に増加。

    ・ステージに関するデータは[stage]フォルダに入っています。

    ・ファイルの構造は大きく分けて５つ、
        １．ベースとなるマップ画像情報
        ２．装飾のマップ画像情報
        ３．マップの当たり判定や特殊ブロックの位置情報
        ４．プレイヤーの位置情報
        ５．敵の位置やステータス、画像のパス等の情報

    ・「stage_temp.js」でより細かいステージの構造がありますので
    　ステージを改造・新規追加したい方はそちらを参考にしてください。

    

【スコア及びランキングについて】

    ・敵の撃破数及び残機数でスコアが発生し、「UserData.json」で保存されます。

    ・保存のタイミングは、「ステージをすべて攻略したとき」または「残機が０になったとき」です。

    ・ランキングではスコアが特に高い１０個のデータが選ばれて、
    　「順位：名前：スコア」の順で表示されます。
    ・20ステージを攻略できたプレイヤーには順位の横に★が出現します。
    　(さらに残機を減らすことなくクリアした場合は★の色が変化します)

    【2023/12/15追加】
    ・味方を追加できる機能及びユーザ名入力機能の実装とともに、
    　ランキングに記録をする条件を追加。
        １．「ユーザ名」を必ず設定すること
        ２．「味方」を追加しないこと
